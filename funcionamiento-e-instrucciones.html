¿Cómo funciona?

Aunque es un programa relativamente sencillo, demás no está explicar su uso, sin dejar de lado algunos recursos necesarios para que funcione correctamente.

Funciones:

El programa actualmente cuenta con solo una función, la cual es convertir audios.

Los formatos soportados son los siguientes:

  1. Formato mp3.
  2. Formato wav.
  3. Formato flac.
  4. Formato opus, útil para la nueva función que se añadirá al programa.
  5. Formato Ogg, nadie usa eso pero igual lo pongo por variedad.

El programa es sencillo, tanto que simplemente al abrir ya te deja en un campo de edición, para escribir o pegar la ruta a tu archivo.

Luego, si continúas tabulando, estará el botón para seleccionar archivos desde el explorador

Después, el siguiente campo será para introducir el nombre del archivo convertido

Los siguientes botones son para convertir audio, y para salir del programa

Al presionar el botón convertir, se abrirá un diálogo donde tendrás que seleccionar el formato deseado

Después de seleccionar, solamente toca esperar el mensaje de de que todo ha salido bien. el archivo se guarda dentro de la carpeta del programa

Posible error y cómo solucionarlo, utilizando el programa add-to-path.

si te encuentras con un error, como que el programa no puede encontrar el archivo solicitado, puedes intentar añadir ffmpeg al path del sistema.

Para hacerlo, tienes 2 opciones

  1. añadirlo con el programa add-to-path:
    1. busca el archivo ejecutable, dentro de la carpeta del programa.
    2. Ejecútalo, preferentemente con privilegios de administrador
    3. Al abrirlo, directamente caerá sobre el botón para seleccionar una carpeta para agregar
    4. En el explorador, busca la carpeta llamada ffmpeg, ubicada dentro de la carpeta _internal, dentro de la misma carpeta del programa
    5. Selecciona la carpeta ffmpeg, y automáticamente se añadirá al path. después de eso, intenta de nuevo convertir un archivo
  2. También puedes hacerlo de forma manual, siguiendo los pasos detallados a continuación
    1. Copia la carpeta ffmpeg como ruta de acceso. generalmente eso puede hacerse si presionas tecla aplicaciones, click derecho sobre la carpeta o shift f10.
    2. Una vez que tengas la ruta en el portapapeles, realiza lo siguiente
    3. En cualquier parte de tu computadora, presiona la tecla del logotipo de windows + la letra r. esto abrirá el cuadro ejecutar
    4. Escribe el siguiente comando, y presiona enter. sysdm.cpl. al hacerlo, caerás en una lista con pestañas
    5. Dentro de esa ventana, navega hasta la pestaña opciones avanzadas, utilizando control + tab para moverse entre las mismas
    6. Dentro de la pestaña opciones avanzadas, busca con tabulador la opción llamada variables de entorno. una vez que la encuentres, presiona enter para acceder
    7. Dentro de esa ventana, busca path, desplazándote con flechas hacia abajo
    8. Cuando ubiques, presiona tabulador hasta el botón editar, o haz el comando ALT + D
    9. caerás en un cuadro para editar. ahí tendrás que pegar la ruta que sacaste anteriormente, de la carpeta ffmpeg. presiona enter y luego ve buscando con tabulador los botones aceptar, hasta cerrar las propiedades del sistema
